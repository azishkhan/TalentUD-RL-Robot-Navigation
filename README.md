# TalentUD RL Robot Navigation in Unity ML-Agents

This project demonstrates reinforcement learning-based robot navigation in Unity using ML-Agents. A humanoid robot agent learns to reach a target while avoiding walls and obstacles.

## Technologies Used

- Unity 2022.3 LTS
- Unity ML-Agents
- Python 3.10
- PyTorch
- PPO
- Behavioral Cloning
- GAIL
- RayPerceptionSensor3D

## Project Features

- Robot navigation environment in Unity
- PPO training with imitation learning
- Simulated LiDAR using RayPerceptionSensor3D
- Reward design and reward exploit correction
- TensorBoard training result analysis
- Sim-to-real transfer learning discussion

## Results

The trained agent showed improved cumulative reward after integrating RayPerceptionSensor3D and fixing the reward function.
